The following is a set of guidelines for contributing to this repository.

## üõ† How to Contribute

- Fork the repository on GitHub.

- Clone the project to your own machine.

- Create a new branch (git checkout -b feature/AmazingFeature).

- Commit your changes to your own branch.

- Push your work back up to your fork.

- Submit a Pull Request so that we can review your changes.

## üß™ Testing

Please ensure that your changes pass the existing tests and write new tests for new functionality.

## Run tests

```
pytest
```

## üé® Coding Style

We use black for code formatting.

Please add comments to complex logic, especially within the Neural Network visualization code.

## üêõ Reporting Bugs

If you find a bug, please create an issue describing:

- What you expected to happen.

- What actually happened.

- Steps to reproduce the issue.
